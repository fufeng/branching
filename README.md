# Feng Fu & Alina Glaubitz --- MATLAB Code for Adaptive Dynamics Analysis

This repository contains MATLAB code used for the adaptive dynamics
analysis presented in our research on evolutionary branching and consistency in
group cooperation.

The scripts implement analytical and numerical procedures for studying
evolutionary singular strategies, stability conditions, and evolutionary
branching. The main output consists of **bifurcation diagrams**
illustrating how equilibrium strategies change with key model
parameters.

## Requirements

-   **MATLAB version:** R2025b (or newer recommended)
-   No additional toolboxes are required beyond standard MATLAB
    functionality.

## Running the Code

1.  Clone or download this repository.
2.  Open **MATLAB (R2025b)**.
3.  Navigate to the repository folder:

``` matlab
cd path/to/repository
```

4.  Run the main script:

``` matlab
bifurcation_selectionGradient_final.m
```

## Output

Running the scripts will generate **bifurcation diagrams** showing the
evolutionary dynamics of cooperation strategies as parameters vary.
Figures will be displayed in MATLAB and can be saved directly from the
figure window.
